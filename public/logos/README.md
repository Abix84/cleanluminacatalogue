# Logos des Entreprises

Placez les logos des entreprises dans ce dossier :

## Fichiers requis

### CleanExpress
- `cleanexpress-logo.png` - Logo principal de CleanExpress

### Lumina Distribution
- `lumina-logo.png` - Logo principal de Lumina Distribution

## Format recommandé

- **Format** : PNG avec transparence
- **Taille** : Minimum 400x400px (pour une bonne qualité)
- **Ratio** : 1:1 (carré) ou adapté au logo original
- **Fond** : Transparent de préférence

## Notes

Si les logos ne sont pas disponibles, l'application affichera un texte de remplacement avec le nom de l'entreprise.
